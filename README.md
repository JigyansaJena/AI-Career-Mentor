# 🧠 CareerMentor AI
> AI-powered career mentoring platform — Agents League Hackathon 2026

## 🚀 What it does
CareerMentor AI helps students and developers land their dream jobs by turning their resume into an actionable career plan in under 60 seconds.

1. 📄 **Resume Upload** — Upload your PDF resume, AI extracts all your skills automatically
2. 📊 **Skill Gap Analysis** — See exactly which skills you're missing for your target role
3. 🗺️ **Learning Roadmap** — Get a personalized AI-generated study plan with resources
4. 🎤 **Mock Interview** — Practice role-specific questions and get instant AI feedback with scores

## 🛠️ Tech Stack
| Layer | Technology |
|-------|-----------|
| Frontend | React, React Router, Axios |
| Backend | Node.js, Express |
| Database | MongoDB |
| AI | Groq (Llama3-8b) |
| Auth | JWT + bcrypt |
| Dev Tool | GitHub Copilot |

## 🎯 Microsoft IQ Integration
Built under the **Creative Apps** track using **GitHub Copilot** for AI-assisted development throughout the entire project.

## ⚙️ Setup

### Backend
```bash
cd career-mentor-server
npm install
# Create .env with: MONGO_URI, JWT_SECRET, GROQ_API_KEY, GROQ_MODEL
npm run dev
```

### Frontend
```bash
cd career-mentor-UI
npm install
npm run dev
```
